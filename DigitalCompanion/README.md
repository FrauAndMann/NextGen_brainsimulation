# Лиза - Цифровой Компаньон

**Полноценная система ИИ с развивающейся личностью**

## Описание

Лиза - это цифровой компаньон с богатой внутренней жизнью:
- **Темперамент** - врождённый, как у человека (сангвиник, холерик, флегматик, меланхолик)
- **Характер** - формируется через опыт, с уменьшающейся пластичностью
- **Нейрохимия** - 120+ нейромедиаторов, влияющих на эмоции
- **Любовь** - треугольник Стернберга (интимность, страсть, обязательства)
- **Память** - эпизодическая, семантическая, эмоциональная

## Возможности

### Эмоциональная система
- PAD модель (Pleasure-Arousal-Dominance)
- Динамические эмоции на основе нейрохимии
- Влияние отношений на эмоциональное состояние

### Система любви
- Треугольная теория Стернберга
- Стиль привязанности (надёжная, тревожная, избегающая)
- Развитие отношений через взаимодействие
- Core memories - важные воспоминания

### Нейрохимия
- Дофамин - мотивация, желание
- Серотонин - настроение, удовлетворение
- Окситоцин - привязанность, любовь
- Кортизол - стресс
- И многие другие (120+)

### Память
- Эпизодическая - события и диалоги
- Семантическая - факты и концепты
- Эмоциональная - яркие моменты
- Кривая забывания Эббингауза

### Голос (TTS)
- Microsoft Edge TTS (высокое качество)
- pyttsx3 (оффлайн)
- Эмоциональная модуляция голоса

## Установка

```bash
# Базовые зависимости
pip install numpy pyyaml

# TTS (голос)
pip install edge-tts

# Опционально
pip install pyttsx3  # альтернативный TTS
```

## Запуск

### Консольный режим
```bash
cd DigitalCompanion
python main.py
```

### Графический интерфейс
```bash
# Простая версия
python gui.py

# Улучшенная версия с аватаром
python gui_v2.py
```

### Параметры запуска
```bash
python main.py --interactive    # Интерактивный режим
python main.py --model llama3.2 # Указать модель Ollama
python main.py --no-tts        # Без голоса
```

## Команды (консольный режим)

| Команда | Описание |
|---------|----------|
| `<текст>` | Отправить сообщение |
| `status` | Полный статус |
| `neuro` | Нейрохимия |
| `trait` | Черты характера |
| `mem` | Состояние памяти |
| `love` | Отношения и чувства |
| `recall <запрос>` | Вспомнить |
| `save` | Сохранить |
| `quit` | Выход |

## Требования для LLM

Для полноценного общения установите [Ollama](https://ollama.ai/):

```bash
# Установка Ollama
# Затем скачайте модель:
ollama pull llama3.2

# Запуск
ollama serve
```

## Архитектура

### Слои личности

1. **Темперамент** (врождённый, не меняется)
   - Свойства нервной системы: сила, уравновешенность, подвижность
   - Влияет на базовые реакции

2. **Нейрохимия** (динамическая)
   - 120+ нейромедиаторов
   - Взаимодействия между ними
   - Влияние на PAD эмоции

3. **Характер** (формируется)
   - Big Five + дополнительные черты
   - Пластичность уменьшается со временем
   - Влияние темперамента как baseline

4. **Эмоции** (текущие)
   - PAD модель
   - Обновление из нейрохимии
   - Влияние на ответы

5. **Отношения** (развиваются)
   - Треугольник любви Стернберга
   - Стиль привязанности
   - Вехи отношений
